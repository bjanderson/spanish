# Spanish

This project is dedicated to resources for learning to speak Spanish.

If you are using Anki, you will find files that you can import into Anki in the Anki folder. Just save each file to your computer and import it into Anki from your computer.

There is also a web application that runs locally on your computer in the spanish_practice folder. There are instructions on how to use it included in that folder. It's experimental, requires you to type Spanish characters, and may go away. Use it if you're interested, but I'm not going to dedicate a lot of time to maintaining it. I prefer to use Anki instead.

I may add other resources to this project in the future, so keep an eye on it.
