# Spanish Practice

Save the index.html file to your computer.

Double-click on it to open it in your browser.

Load any of the data files in this folder.

You will be presented with text in one languge, and have to type in the translated text that matches it.

If you don't know how to type Spanish characters (Ñ, ñ, á, é, í, ó, ú, ü, ¿, ¡) you can copy/paste them from here, or ask Google for instructions on how to do that on your particular operating system.

You can change the data in the app using the Manage tab. Make sure to Save (download) your changes if you want to keep them.

Also, you can export the data from this app to Anki compatible text files, which you can then import into Anki.
