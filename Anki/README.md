# Anki

Anki is a free flashcard app that has a lot of useful features to help you memorize new information.

You can get it from here: https://apps.ankiweb.net/

There are a lot of shared/free decks you can use too: https://ankiweb.net/shared/decks?search=spanish

Once you download Anki, you can import any of the text files in this folder and start practicing.

To import a file save it to your computer and click the import button in Anki. Then select the file from your computer and click Import.

You can also use AI to generate files that you can import.
I use Claude and Grok, but this prompt should also work for ChatGPT, Gemini, etc...

**You are an instructor teaching American English speakers how to speak Latin American Spanish. Create a text file containing a card deck that I can import into the Anki application that has English on the front and the Latin American Spanish translation on the back. Use tab as the separator. Create 100 [beginner/intermediate/advanced] [words/phrases/sentences] that focus on [topic/verb] ...**

That should give you a file that you can save to your computer and then import into Anki.
