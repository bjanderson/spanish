# Anki

Anki is a free flashcard app that has a lot of useful features to help you memorize new information.

You can get it from here: https://apps.ankiweb.net/

There are a lot of shared/free decks you can use too: https://ankiweb.net/shared/decks?search=spanish

Once you download Anki, you can import any of the text files in this folder and start practicing.

To import a file save it to your computer and click the import button in Anki. Then select the file from your computer and click Import.
